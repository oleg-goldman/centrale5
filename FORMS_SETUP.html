Note: The current forms in index.html are configured for server-side processing. Since this is now a standalone static site, you need to configure form submissions.

üöÄ Option 1: Formspree (Recommended)

Free tier: 50 submissions/month

  1. Go to formspree.io and create a free account
  2. Create a new form and get your form endpoint
  3. In index.html, find all forms and update the action attribute:
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
Benefits:
  • Email notifications
  • Spam protection
  • File uploads support
  • Easy integration

üì® Option 2: Vercel Contact Forms

Free tier: 100 submissions/month

  1. Install Vercel CLI: npm i -g vercel
  2. Deploy your site: vercel
  3. Enable Contact Forms in your Vercel dashboard
  4. Update forms in index.html:
<form data-static-form-name="contact">
    <input type="hidden" name="form-name" value="contact" />
    <!-- rest of your form fields -->
</form>

üîß Option 3: Custom API (Advanced)

Create a Vercel Serverless Function:

  1. Create /api/contact.js file:
export default async function handler(req, res) {
    if (req.method === 'POST') {
        const { name, phone, email } = req.body;
        
        // Send email using SendGrid, Mailgun, etc.
        // Or save to database
        
        res.status(200).json({ status: 'ok' });
    } else {
        res.status(405).json({ error: 'Method not allowed' });
    }
}
  1. Update form action in index.html:
<form action="/api/contact" method="POST">
